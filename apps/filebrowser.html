Description
What is this? A really simple file browser for iOS, watchOS & tvOS.
It allows you to browse the iOS filesystem and retrieve files using iMessage or AirDrop.

Note: The watchOS target requires a number of steps to build. You want to select the 'NanoFileBrowser-Package' target - this builds all the requisite things, and patches the built WatchKit app to load natively. You will need to manually install the FileBrowser app on your iOS device in the Devices window - build & run will not work in this case.

Visit the official project page to learn more.
