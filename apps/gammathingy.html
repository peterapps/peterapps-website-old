Description
Gamma Thingy changes the screen gamma on iOS, no jailbreak required.
With the new Night Shift feature on iOS 9.3, this is pretty much obsolete.

If you find the display glitching while Gamma Thingy is enabled, ensure that you have disabled "Reduce White Point" in Settings > General > Accessibility > Increase Contrast.
