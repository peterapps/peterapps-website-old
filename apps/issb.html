Description
iSSB is a mobile version of the popular Super Smash Bros. Brawl!
This application was developed by Luis Finke (aka @lufinkey), a programmer and student with the focus of creating a game based on a highly popular series that you would want to play. Several versions of iSSB can be played on numerous devices such as Windows, Mac, Linux and Online. The iOS version can be found on both Jailbreak and non-Jailbreak devices.
