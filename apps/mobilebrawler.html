Description
MobileBrawler is a mobile version of the popular Super Smash Bros. Brawl!
This project is a re-write of the older iSSB repo (http://github.com/lufinkey/iSSB). MobileBrawler is designed to provide an experience similar to Super Smash Bros. on iOS devices.

This project uses the SDL game library and is written in C++. The Windows project is for Visual Studio 2015, and the iOS project is for Xcode.
