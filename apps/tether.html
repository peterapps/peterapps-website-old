Description
Tether your unjailbroken iPhone to your desktop computer via USB for free!
his was inspired by iProxy, but I always had trouble getting the damn ad-hoc wifi network to work reliably.

Right now this is mainly proof-of-concept... but it works! It will let you create a SOCKS proxy running on your iPhone and mirror the socket to your local machine over USB via usbmuxd.

Note: You also need to install Tether on your Mac and set up the SOCKS proxy settings in Firefox. Contact Peter if you have any questions or go to the official project page.
